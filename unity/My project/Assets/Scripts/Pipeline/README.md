# Pipeline 化工厂管道培训模块

## 文件说明

| 文件 | 用途 |
|------|------|
| `PipelineBuilder.cs` | 核心管道模型构建器 —— 程序化生成直管、弯头、三通、四通、U形管、阀门段、压力表段、流量计段、法兰、支架、急停面板等 |
| `PipelineChemicalSceneBuilder.cs` | 化工厂管道培训场景搭建器 —— 继承 PipelineBuilder，按 10 个培训步骤布局完整场景 |
| `PipelineTrainingManager.cs` | 培训流程管理器 —— 管理 10 步培训流程（Guide/Practice 双模式）|
| `PipelineSceneRuntime.cs` | 运行时控制器 —— 键盘交互、物理模拟（压力/流量）、仪表驱动 |
| `PipelineZoneTrigger.cs` | 区域触发器 —— 巡检点自动检测玩家进入 |

## 10 步培训流程

| 步骤 | 名称 | 操作类型 |
|------|------|----------|
| 1 | 安全装备确认 | 按钮确认 |
| 2 | 系统状态巡检 | 区域触发 |
| 3 | 读取初始压力表 P1 | 仪表观察 |
| 4 | 开启进口主阀门 V1 | 阀门旋转 |
| 5 | 监测流量计 F1 | 仪表观察 |
| 6 | 调节中间控制阀 V2 | 阀门旋转 |
| 7 | 检查中段压力表 P2 | 仪表观察 |
| 8 | 开启出口阀门 V3 | 阀门旋转 |
| 9 | 急停功能测试 | 按钮按下+复位 |
| 10 | 系统关停与记录 | 综合操作 |

## 管道模型统一规范

| 参数 | 数值 |
|------|------|
| 管道外径 | 0.30 m |
| 法兰外径 | 0.50 m |
| 法兰厚度 | 0.08 m |
| 弯头弯曲半径 | 0.40 m |
| 阀门手轮直径 | 0.70 m |
| 压力表表盘直径 | 0.25 m |
| 流量计本体宽度 | 0.35 m |

## 使用方式

### Editor 中生成场景

```
Tools > Pipeline > Create Static Chemical Pipeline Scene In Current Scene
Tools > Pipeline > Validate Chemical Pipeline Scene
```

### 运行模式

- **Guide 模式**：系统逐步引导玩家完成 10 个步骤
- **Practice 模式**：玩家自由操作，所有步骤同时开放

### 键盘操作

| 按键 | 操作 |
|------|------|
| WASD | 移动 |
| Q/E | 旋转阀门（靠近阀门时）|
| F | 交互确认（按钮、巡检点）|
| Space | 急停按钮 |
| R | 急停复位 |
